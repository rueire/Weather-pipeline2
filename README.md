# Weather-pipeline2
Weather pipeline (open-meteo: https://open-meteo.com/en/docs) using python 

when in directory create venv and activate it:<br>
python -m venv venv<br>
venv\Scripts\activate

then install requirements:<br>
pip install -r requirements.txt

to see whole week's temperatures:<br>
python pipeline.py

to see next day's temperatures:<br>
python tmr_pipeline.py
